## Описание
Брендбук и фирменный стиль для Лены Алпатовой (нутрициолог, ПП-рационы). Стиль: Art Deco, sage green + gold. Маскот — фея с венчиком (Fairytrue.svg — единственный исходник). В будущем — продающий сайт.

## Стек / Инструменты
- HTML/CSS/JS (brandbook.html — интерактивный брендбук)
- SVG (Fairytrue.svg — единственный исходник логотипа-феи)
- Google Fonts: Forum, Cormorant Garamond

## Файлы
- `brandbook.html` — основной файл брендбука (палитра, типографика, логотип, визитка, наклейка, правила, скачивание)
- `Fairytrue.svg` — единственный SVG-исходник логотипа феи (позитивный силуэт)
- `fairyalpatova.html` — старый черновой файл (raw SVG)
- `poe-preview (2).html` — предыдущая версия брендбука (заменена brandbook.html)
- JPG-файлы — фото визитки, наклейки, логотипа
